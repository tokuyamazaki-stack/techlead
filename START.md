## 起動手順

1. VS Code のターミナルを開く（Ctrl + @）
2. 以下を貼り付けて Enter

```
cd C:\Users\admin\Desktop\techlead
npm run dev
```

3. ブラウザで http://localhost:3000 を開く

## 今どこまで作ったか（2026/04/13時点）

### ✅ 完成している機能
- **Sansanデータ取込（AI解析）**
  - Sansanのページを Ctrl+A → Ctrl+C でコピーするだけ
  - GroqのAIが自動で企業データだけ抽出
  - リスト名をつけて取込める（例：製造業 300億以上）
- **リスト管理UI**
  - 複数のリストをカード形式で表示
  - クリックで切り替え、✕で削除
- **コール結果記録**
  - 企業をクリック → ワンクリックで結果記録
  - タグ（商材・課題・興味ポイント）、次回連絡日、担当者
- **日報自動生成**
  - 本日のコール結果を集計してテキストコピー
- **分析画面**
  - 業種別アポ率、担当者別成績、7日間トレンド

## 環境設定

- Groq APIキー：`.env.local` に `GROQ_API_KEY=...` で設定
- ⚠️ APIキーはチャットに貼らないこと（自動で無効化される）
- キーを変えたらサーバー再起動が必要（Ctrl+C → npm run dev）

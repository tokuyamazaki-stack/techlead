## 今どこまで作ったか（2026/04/14時点）

### ✅ 完成している機能

- **Sansanデータ取込（AI解析）**
  - Sansanのページを Ctrl+A → Ctrl+C でコピーするだけ
  - GroqのAIが自動で企業データを抽出（会社名・電話・業種・従業員数・売上・住所）
  - リスト名＋絞り込み条件（業界・決算月・売上規模）をつけて取込める

- **リスト管理UI**
  - 複数のリストをカード形式で表示
  - カードに「業界（大きく）・決算月・売上規模・件数」を表示
  - クリックで切り替え、ダブルクリックまたは ✎ で名前編集、✕で削除

- **コールリスト表示**
  - 会社名・電話番号・業種・従業員数・売上・住所・結果・次回連絡日を表示
  - フィルター（すべて／未コール／本日ネクスト／結果別）＋検索
  - 本日ネクストの企業を上部にハイライト表示

- **コール結果記録**
  - 企業をクリック → ワンクリックで結果記録
  - タグ（商材・課題・興味ポイント）、次回連絡日、担当者
  - アポ獲得時にGoogleカレンダーのiframeを表示

- **担当者設定（マイ設定）**
  - ヘッダーのアバターボタンから設定
  - 担当者名を一度設定すればコール記録に自動入力
  - GoogleカレンダーURL（予約スケジュール）を登録可能

- **日報自動生成**
  - 本日のコール結果を集計してテキストコピー

- **分析画面**
  - 業種別アポ率、担当者別成績、7日間トレンド

## 環境設定

- Groq APIキー：`.env.local` に `GROQ_API_KEY=...` で設定
- ⚠️ APIキーはチャットに貼らないこと（自動で無効化される）
- キーを変えたらサーバー再起動が必要（Ctrl+C → npm run dev）

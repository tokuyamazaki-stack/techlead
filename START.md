## 今どこまで作ったか（2026/04/16時点）

### 作り始め：2026年4月13日（3日で公開まで完了）

### 完成している機能

- **Sansanデータ取込（AI解析）**
  - SansanのページをCtrl+A → Ctrl+Cでコピーするだけ
  - GroqのAIが自動で企業データを抽出（会社名・電話・業種・従業員数・売上・住所）
  - リスト名＋絞り込み条件（業界・決算月・売上規模）をつけて取込める

- **リスト管理UI**
  - 複数のリストをカード形式で表示
  - クリックで切り替え、ダブルクリックで名前編集、✕で削除

- **コールリスト表示**
  - 会社名・電話番号・業種・従業員数・売上・住所・結果・次回連絡日を表示
  - フィルター（すべて／未コール／本日ネクスト／結果別）＋検索
  - 本日ネクストの企業を上部にハイライト表示

- **コール結果記録**
  - 企業をクリック → ワンクリックで結果記録
  - タグ（商材・課題・興味ポイント）、次回連絡日、担当者、NG理由
  - アポ獲得時にGoogleカレンダーのiframeを表示

- **担当者設定（マイ設定）**
  - 担当者名・カレンダーURL・電話番号・メールアドレス
  - 日報フォームURL＋Googleフォーム自動入力設定

- **日報自動生成**
  - 本日のコール結果を集計してテキストコピー
  - 担当者別集計・アポ獲得企業一覧
  - Googleフォームへの自動入力送信

- **分析画面**
  - 業種別アポ率、担当者別成績、7日間トレンド
  - 月間目標に対する進捗

- **デモデータ**
  - 初回起動時に3リスト・30社・28コール履歴が自動表示
  - 設定 → デモデータにリセット でいつでも戻せる

## 環境設定

- Groq APIキー：`.env.local` に `GROQ_API_KEY=...` で設定
- VercelにもGROQ_API_KEYを登録済み（Vercel CLI経由）
- ⚠️ APIキーはチャットに貼らないこと
- キーを変えたらサーバー再起動が必要（Ctrl+C → npm run dev）

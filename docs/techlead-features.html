起点 — まずここから

AI通話録音を動かす

日本の電話番号を取得してデータを溜め始める。全ての機能はここから始まる。通常の架電をするだけで自動で蓄積される。

⚡
Phase 1 — 1〜2ヶ月後

通話中リアルタイムコーチング

相手が話している最中に、画面にベストな返しが表示される。「こう来たらこう返す」をAIがその場で提案。新人でもトップと同じ対話ができる。

🏆
Phase 1 — 1〜2ヶ月後

勝ちパターン自動学習

アポを一番取っている人の返し方・切り出し方をAIが自動抽出。チーム全員の画面に「この返しでアポ率23%」が表示されるようになる。属人化がチームの武器になる瞬間。

💬
Phase 2 — 3〜6ヶ月後

スクリプト自動生成 & A/Bテスト

AIとの対話だけでスクリプト3パターンを自動生成。どの切り出しが刺さるか、データで比較できる。マネージャーの経験則が不要になる。

📊
Phase 3 — 6〜12ヶ月後

個人別コーチングダッシュボード

「この人はクロージングが弱い」「この切り出しは使っていない」をAIが自動分析。マネージャーが感覚ではなくデータで指導できる。
